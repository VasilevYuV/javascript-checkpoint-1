# Веломастерская «Велозар»

Лендинг на **React** + **Vite**. Статика и шрифты подключаются локально (без CDN).

## Требования

- **Node.js** 18+ (рекомендуется LTS)
- **npm** (идёт вместе с Node)

## Установка зависимостей

```bash
npm install
```

## Основные команды

| Команда | Описание |
|--------|----------|
| `npm run dev` | Запуск dev-сервера с hot reload (по умолчанию [http://localhost:5173](http://localhost:5173)) |
| `npm run build` | Сборка production-версии в папку `dist/` |
| `npm run preview` | Локальный просмотр собранного `dist/` (после `npm run build`) |
| `npm run lint` | Проверка кода ESLint |

## Кратко

1. `npm install` — один раз после клонирования.
2. `npm run dev` — разработка.
3. `npm run build` — подготовка файлов для выкладки; результат в `dist/`.
